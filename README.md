# Capstone 2 Sales Analysis Project

## Project Overview

This project analyzes sales performance for assigned sales territories using Python and Jupyter Notebook.

The analysis focuses on:
- sales trends
- store performance
- customer purchasing behavior
- product performance
- marketing recommendations

The project was completed as part of the Year Up United Data Analytics program.

---

## Technologies Used

- Python
- Pandas
- Matplotlib
- Jupyter Notebook
- Git and GitHub

---

## Datasets Used

The project uses the following datasets:

- StoreSales.csv
- StoreDetail.csv
- Products.csv
- ProductCategories.csv
- customer_list.csv

---

## Analysis Completed

The notebook includes analysis for:

1. Territory managers and store information
2. Monthly sales revenue analysis
3. Store performance rankings
4. Top customer identification
5. Product sales and transaction analysis
6. Final business recommendations

---

## Visualizations

The project includes:
- Monthly revenue trend charts
- Product revenue comparison charts

---

## Key Findings

- Florida generated stronger overall sales performance across most months.
- Several products consistently generated high revenue and transaction volume.
- South Carolina showed opportunities for future marketing growth.
- Customer rewards participation contributed to identifying top-performing customers.
